## 💡 特征工程建议

### 1. 创建组合特征

```python
# 通信活跃度
df['communication_activity'] = df['tx_event_rate'] + df['rx_event_rate']

# 移动活跃度
df['mobility_activity'] = df['avg_speed'] * df['total_distance']

# 网络重要性
df['network_importance'] = df['avg_degree'] * df['total_flows']

# 通信平衡度
df['communication_balance'] = 1 - abs(df['tx_ratio'] - 0.5) * 2

# 位置变化度
df['position_change'] = df['std_x'] + df['std_y'] + df['std_z']
```

### 2. 特征归一化

```python
from sklearn.preprocessing import StandardScaler, MinMaxScaler

# 选择数值特征
numeric_features = df.select_dtypes(include=[np.number]).columns
numeric_features = [f for f in numeric_features if f not in ['nodeId']]

# 标准化
scaler = StandardScaler()
df_scaled = df.copy()
df_scaled[numeric_features] = scaler.fit_transform(df[numeric_features])

# 或归一化到[0,1]
minmax_scaler = MinMaxScaler()
df_minmax = df.copy()
df_minmax[numeric_features] = minmax_scaler.fit_transform(df[numeric_features])
```

### 3. 特征选择


## 📝 快速使用示例

```python
import pandas as pd
import numpy as np

# 加载特征数据
df = pd.read_csv('node_features_all.csv')

# 1. 查看所有特征
print("所有特征:", df.columns.tolist())
print(f"特征总数: {len(df.columns) - 2}")  # 减去nodeId和dataset

# 2. 选择关键特征
key_features = [
    'avg_degree', 'src_avg_pdr', 'avg_speed', 
    'position_stability', 'total_flows', 'tx_event_rate'
]
df_key = df[['nodeId', 'dataset'] + key_features]

# 3. 按数据集分组统计
for dataset, group in df.groupby('dataset'):
    print(f"\n{dataset}:")
    print(f"  节点数: {len(group)}")
    print(f"  平均度数: {group['avg_degree'].mean():.2f}")
    print(f"  平均PDR: {group['src_avg_pdr'].mean():.3f}")

# 4. 识别关键节点（高连接度+高PDR）
df['node_score'] = df['avg_degree'] * df['src_avg_pdr']
top_nodes = df.nlargest(10, 'node_score')[['nodeId', 'dataset', 'node_score']]
print("\n关键节点:")
print(top_nodes)
```
